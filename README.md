# day-word

This is a simple Node.js project that serves as a template for building applications. 

## Project Structure

```
day-word-project
└── wordle-solver
|   ├── public
|       └── index.html
|   ├── src
|       ├── index.js
|       ├── solver.js
|       └── utils
|           └── clues.js
|   ├── package.json
|   └── README.md
```

## Installation

To install the necessary dependencies, run:

```
npm install
```

## Usage

To start the application, use the following command:

```
npm start
```
